# Overview

An HTML5 rebuild of the Cartoon Network Video application from 2006, originally developed by Dreamsocket.

## Installation

Setup your favorite localhost and just dump the files into htdocs

## Adding Content

Click on the Shows or Extras page, and in the breadcrumbs at the top
you'll see a plus icon, click it to access the administration panel.
Click the New Collection button to open the collection creation panel

NOTE: Any changes you make must be saved via the export tab and replacing 
the XML manually, as JavaScript cannot make changes to your local disk files.

Make sure any content you want to add being sourced from and is in protected_auth!

## Show Headers

Each header has 1px of white space at the top, and 7 px at the bottom.
And only the logos can touch outside of both the top and bottom of the header. 
Keep this in mind when designing cohesive headers.

## Watermarks

Sorenson Squeeze was used back in the day to apply watermarks.
Using the included graphic "bug.png", use the following presets
for Sorenson's watermark filter.

Opacity: 30
Feathering: 0
Position: Bottom Right
Scale: 20

Export your videos with F8_FLV_VP6_768bps_360p
